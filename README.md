# 🤖 Auto Job Finder & Apply

A fully autonomous AI agent that discovers jobs, evaluates them against your profile, generates tailored CVs and cover letters using AI, auto-applies, and sends you WhatsApp notifications — all on autopilot.

## ✨ Features

- **AI-Powered CV & Cover Letters** — Each CV/cover letter is generated by AI (Pollinations, free, no API key) specifically tailored to the job and company
- **Multi-Source Job Discovery** — Scrapes LinkedIn, RemoteOK, Remotive, WeWorkRemotely, Jobspresso, and **Freelancer.com** (freelance gigs/projects)
- **Smart Job Evaluation** — Scores each job against your profile (skills, experience, location preferences)
- **WhatsApp Notifications** — Sends you a WhatsApp message every time it applies to a job (no paid API, uses WhatsApp Web automation)
- **Proof of Application** — Captures a screenshot of the job page after applying, so you can verify the application was actually made
- **Web Dashboard** — Set up profile, connect WhatsApp, view all applications (with proof screenshots) and activity
- **Fully Autonomous** — Runs automatically every 6 hours. You set up once, it does the rest.
- **No API Keys Needed** — AI (Pollinations) and WhatsApp (Web automation) both work without any paid API keys

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/Hadi990op/Auto-job-Finder-Apply.git
cd Auto-job-Finder-Apply

# Run setup (as root)
sudo ./setup.sh
```

That's it. The dashboard will be live at `http://localhost:9300`.

### One-Line Install (no clone needed)

```bash
curl -sL https://raw.githubusercontent.com/Hadi990op/Auto-job-Finder-Apply/main/quick-install.sh | sudo bash
```

## 📋 Setup After Install

1. **Open the dashboard** → `http://localhost:9300`
2. **Set up your profile** → Go to `/profile` — enter your name, skills, experience, job titles, preferred locations, and auto-apply threshold
3. **Connect WhatsApp** → Go to `/whatsapp` — enter your phone number, get a pairing code, enter it on your phone's WhatsApp → Settings → Linked Devices
4. **Upload your base CV** (optional) → Go to `/profile` — upload your existing CV. The AI will use it as context to generate better tailored CVs.
5. **Done!** The agent runs automatically every 6 hours. You'll get WhatsApp notifications for every application.

## 🤖 AI Details

- **Provider**: [Pollinations AI](https://text.pollinations.ai) — free, no API key, no registration
- Uses OpenAI-compatible API for CV and cover letter generation
- Falls back to template-based generation if AI is unavailable
- Each CV/cover letter is uniquely tailored to the specific job posting

## 📱 WhatsApp Details

- **Method**: Playwright (headless Chromium) automating WhatsApp Web
- **No API key, no paid services** — pure web automation
- Pair once with a code (like WhatsApp Desktop), session persists
- Sends formatted notifications with job title, company, fit score, and link

## 🤝 How It Works

1. **You set up your profile once** — name, skills, experience, job titles you want, locations you prefer
2. **You connect WhatsApp once** — pair with a code, just like WhatsApp Desktop
3. **The agent runs every 6 hours automatically:**
   - Discovers jobs from 6 sources (LinkedIn, RemoteOK, Remotive, WWR, Jobspresso, Freelancer.com)
   - Evaluates each job against your profile (keyword matching + fit score)
   - For matching jobs: AI generates a tailored CV and cover letter
   - Records the application in the database
   - Captures a screenshot of the job page as proof of application
   - Sends you a WhatsApp notification with job details
4. **You see everything in the dashboard** — all jobs, applications (with proof screenshots), activity logs
